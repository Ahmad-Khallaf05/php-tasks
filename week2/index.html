What is a Database?

A database is an organized collection of data that is stored and accessed electronically. Databases are designed to manage and store large amounts of information efficiently, allowing users to retrieve, update, and manipulate data easily. They can be used to store various types of data, including text, numbers, and multimedia files, and are essential for applications that require persistent data storage, such as websites, software programs, and business systems.

SQL Basics

SQL, or Structured Query Language, is a standard programming language used for managing and manipulating relational databases. It is essential because it provides a systematic way to create, retrieve, update, and delete data within a database. SQL commands allow users to perform complex queries, join tables, and enforce data integrity, making it a critical tool for database management and operations.

What is MySQL?

MySQL is an open-source relational database management system (RDBMS) based on SQL. It is known for its reliability, performance, and ease of use. MySQL supports various storage engines, robust data security features, and a wide range of functions and tools for database management. It is commonly used for web applications, enterprise solutions, and data warehousing due to its scalability and compatibility with various platforms.

Numeric Data Types in MySQL

  • INT: A standard integer data type. Example: age INT
  • FLOAT: A floating-point number with precision. Example: price FLOAT
  • DOUBLE: A double-precision floating-point number. Example: salary DOUBLE
  • DECIMAL: A fixed-point number with a specified precision and scale. Example: balance DECIMAL(10,2)

String Data Types in MySQL

  • CHAR: A fixed-length string. Example: code CHAR(5)
  • VARCHAR: A variable-length string. Example: name VARCHAR(100)
  • TEXT: A large variable-length string. Example: description TEXT
  • BLOB: A binary large object for storing binary data. Example: image BLOB

Date and Time Data Types in MySQL

  • DATE: A date value in 'YYYY-MM-DD' format. Example: birthdate DATE
  • DATETIME: A date and time value in 'YYYY-MM-DD HH:MM:SS' format. Example: created_at DATETIME
  • TIME: A time value in 'HH:MM:SS' format. Example: start_time TIME
  • TIMESTAMP: A timestamp value for tracking changes. Example: updated_at TIMESTAMP

What is a Primary Key?

A primary key is a unique identifier for each record in a database table. It ensures that each record can be uniquely identified and retrieved. Primary keys are important because they enforce entity integrity, preventing duplicate records and ensuring that each record is distinct.

NULL and NOT NULL Constraints

NULL constraints allow a column to have no value, whereas NOT NULL constraints require a column to always have a value. For example, a column defined as name VARCHAR(100) NOT NULL means the name field must always contain a value, while email VARCHAR(100) allows the email field to be empty.

What is AUTO_INCREMENT?

AUTO_INCREMENT is a MySQL feature that automatically generates a unique number for each new record in a table, typically used for primary key columns. For example, id INT AUTO_INCREMENT PRIMARY KEY ensures that each new record has a unique ID value.

Creating a Database

The SQL command to create a new database named StudentDB is:

CREATE DATABASE StudentDB;

Creating a Table

The SQL command to create a table named Students with the specified columns is:


CREATE TABLE Students (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(100),
  enrollment_date DATE
);

Insert, Update, Delete

Insert Data: The SQL command to insert a new student record is:

INSERT INTO Students (name, email, enrollment_date) VALUES ('John Doe', 'john.doe@example.com', '2023-01-15');

Update Data: The SQL command to update the email of the student with id = 1 is:

UPDATE Students SET email = 'new.email@example.com' WHERE id = 1;

Delete Data: The SQL command to delete the student record with id = 1 is:

DELETE FROM Students WHERE id = 1;

Using a Database

The SQL command to select and use the StudentDB database is:

USE StudentDB;
